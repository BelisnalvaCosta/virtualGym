# virtualGym
Academia virtual
Este projeto feito na IDE IntelliJ, linguagem java 11 com Spring DATA JPA. 
O propietário "funcionário" da Academia pode Cadastrar Clientes, e seus dados com segurança.

Dependências:
Foi utilizada o Maven
Dependências:
.Spring Web
.Spring Data Jpa
.Hibernate validator
.Lombok
.PostgreSQL
.Postman
